Projects from the 100 Days of Code Python Bootcamp course.

---

## 🔰 Projects Overview

### [Day-1](./Day-1) - Quiz Brain Application
A terminal-based quiz game that tests your knowledge. The project demonstrates:
- **Object-Oriented Programming (OOP)** concepts
- Creating custom classes (`question`, `QuizBrain`)
- Working with data structures (lists, dictionaries)
- Random shuffling of questions
- Score tracking

**Files:**
- `main.py` - Main quiz application
- `question_model.py` - Question class
- `quiz_brain.py` - QuizBrain class for managing quiz logic
- `data.py` - Quiz questions data

---

### [Day-2](./Day-2) - Hirst Painting Generator
An artistic project that generates paintings inspired by Damien Hirst's dot paintings using Python's Turtle graphics library.
- **Key Concepts:**
  - Turtle graphics and drawing
  - Color extraction from images using colorgram
  - Random color selection
  - Coordinate system and positioning
  - Nested loops for pattern generation

**Files:**
- `main.py` - Main painting generator script
- `hirst_painting.py` - Painting functions
- `img.jpg` - Reference image for color extraction

---

### [Day-3](./Day-3) - Turtle Graphics Games
Two fun turtle graphics projects demonstrating animation and game mechanics:

#### Etch-a-Sketch
- Control a turtle with keyboard inputs to draw
- **Concepts:** Event listeners, turtle movement, keyboard input handling

#### Turtle Race Game
- Watch turtles race across the screen with random movements
- Betting simulation game
- **Concepts:** Turtle animation, random number generation, conditional logic

**Files:**
- `etch-a-sketch.py` - Drawing/sketching application
- `turtle_race_game.py` - Racing simulation game

---

### [Day-4-5](./Day-4-5) - Snake Game
A classic Snake game implementation using Object-Oriented Programming and Turtle graphics.
- **Key Concepts:**
  - Game loop and animation
  - Collision detection
  - Score tracking and high scores
  - Class inheritance
  - Game state management

**Features:**
- Playable snake game with keyboard controls
- Food spawning and eating mechanics
- Score board with high score tracking
- Game over detection

**Files:**
- `main.py` - Main game loop
- `snake_body.py` - Snake class and movement logic
- `food.py` - Food spawning logic
- `score_board.py` - Score tracking and display
- `highscore.txt` - Persistent high score storage

---

## 🛠 Technologies Used
- **Python 3**
- **Turtle Graphics** - For graphics and animations
- **colorgram** - For color extraction from images
- **File I/O** - For data persistence (high scores)
